Welcome to the PyAutoGrader documentation. PyAutoGrader is a free automated Python grading tool for educators. Students run the grader on their own machines, getting instant feedback on their code.

## Quick Links

- [Overview](/docs/offline/overview/) - What PyAutoGrader does and how it works

- [Instructor Setup](/docs/offline/instructor-setup/) - Create assignments and configure tests
- [Test Types Reference](/docs/offline/test-types-reference/) - Full catalog of 55+ test types
- [Student Usage](/docs/offline/student-usage/) - How students run the grader

## For Instructors

Start with the [Overview](/docs/offline/overview/) to understand the workflow, then [Download and Install](/docs/offline/download-and-install/) to get the applications, and follow [Instructor Setup](/docs/offline/instructor-setup/) to create your first assignment.

## For Students

Your instructor will provide you with a `.agr` assignment file. Download the student app from the [Student Downloads](/student/offline/) page, load the file, and run the grader against your code. See [Student Usage](/docs/offline/student-usage/) for a full walkthrough.
