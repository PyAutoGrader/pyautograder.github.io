Welcome to the PyAutoGrader documentation. PyAutoGrader is a free automated Python grading tool for educators. Students run the grader on their own machines, getting instant feedback on their code.

## Quick Links

- [Overview](/docs/offline/overview/) - What PyAutoGrader does and how it works
- [Instructor Setup](/docs/offline/instructor-setup/) - Create assignments and configure tests
- [Student Usage](/docs/offline/student-usage/) - How students run the grader
- [Building the Executable](/docs/offline/building-executable/) - Package assignments for distribution

## For Instructors

Start with the [Overview](/docs/offline/overview/) to understand the workflow, then follow [Instructor Setup](/docs/offline/instructor-setup/) to create your first assignment.

## For Students

Your instructor will provide you with a `.agr` assignment file. Download the student app from the [Student Downloads](/student/offline/) page, load the file, and run the grader against your code.

## Documentation is a Work in Progress

These docs are being expanded. If something is missing, use the [Contact](/contact/) page to ask - or to request a custom test type.
